# Product

## Register

product

## Users

Engineering teams at Buk (company domain `@buk.*`). Developers and tech leads who assign PR reviews daily, often multiple times per session. Context: open on a second monitor or a tab alongside their IDE, glanced at quickly while in flow. The primary task on any screen is "assign this PR to the right person, fast."

## Product Purpose

La Lista is a round-robin PR review assignment tool. It removes the cognitive overhead of deciding who reviews what by automating rotation while respecting reviewer availability. Success looks like: someone pastes a PR URL, hits assign, and gets back to coding in under 10 seconds.

## Brand Personality

Precise. Opinionated. Lightweight.

## References

Linear — clean information hierarchy, confident typography, dark-by-default, every pixel has a reason. Not sterile but never decorative.

## Anti-references

AI slop: gradient text, side-stripe borders on cards, hero-metric templates (big number + gradient + label), identical card grids, glassmorphism used decoratively, modal as first thought. Generic SaaS dashboards that feel assembled from a template kit.

## Design Principles

1. **Speed over ceremony.** The primary action (assign PR) must be reachable with zero friction. Everything else yields to it.
2. **Density with breathing room.** Dev tools need information density, but La Lista is used in bursts, not stared at for hours — give the layout enough air to read at a glance.
3. **Confident defaults.** Don't ask the user what they want if you can infer it. Controls appear when needed, not upfront.
4. **No decoration for decoration's sake.** Every visual element communicates something. If it doesn't, it's gone.
5. **Consistent enough to be invisible.** Spacing and layout should be predictable so the user never has to think about the interface.

## Accessibility & Inclusion

Standard WCAG 2.1 AA. Support both dark and light themes (system default). Keyboard navigation for primary actions (already partially implemented). Respect `prefers-reduced-motion`.
